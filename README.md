# Outa Music Bot

A minimalist Discord music bot written in Python using `discord.py` and `yt-dlp`.

## Important Note
All code in this repository was entirely written and generated by **Gemini AI**. The creator/owner of this repository simply acted as the director, giving instructions and commands to the AI to build and modify its features.

## Features
- Play music or playlists directly from YouTube (`!play`).
- Automated music queue with a *pre-fetching* system for gapless playback between songs.
- View detailed information about the currently playing track (`!player` / `!np`).
- Playback controls (`!pause`, `!resume`, `!skip`, `!stop`, `!seek`, `!forward`, `!back`, `!volume`).
- Queue management (`!shuffle`, `!move`, `!drop`, `!clear`).
- Disconnect the bot (`!quit`).
- **DJ System**: Only the user who requested the first track (or a Server Admin) has permission to control the bot.

## Requirements
- Python 3.9+
- **FFmpeg** must be installed on your operating system.
- Python packages (see `requirements.txt`):
  - `discord.py`
  - `yt-dlp`
  - `python-dotenv`
  - `PyNaCl`

## How to Run
1. Clone or download this repository.
2. Create a file named `.env` in the root directory and fill it with your Discord Bot Token:
   ```env
   BOT_TOKEN=your_bot_token_here
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Start the bot:
   ```bash
   python main.py
   ```

Alternatively, you can run the bot using Docker via `docker-compose up -d`.

**Default Prefix**: `!`
Type `!help` in a Discord channel to view the full list of available commands.
