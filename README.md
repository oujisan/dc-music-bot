# Outa Music Bot

A Discord music bot written in Python using `discord.py` and `yt-dlp`.

## Important Note
All code in this repository was entirely written and generated by **Gemini AI**. The creator/owner of this repository simply acted as the director, giving instructions and commands to the AI to build and modify its features.

## Features
- **Smart Queue & Playback**: Play music or playlists directly with an automated *pre-fetching* system for gapless playback.
- **Dedicated Search Engines**: Use `!play` and `!show` to fetch official audio from **YouTube Music**, or use `!ytplay` and `!ytshow` to search **Standard YouTube** for Music Videos and covers.
- **Interactive Search**: Search for tracks and view results in an interactive dropdown menu (`!show` / `!ytshow`) before playing or adding to the queue.
- **Lyrics & Subtitles**: Fetch video subtitles directly (`!subs`) or fetch/translate track lyrics (`!lyrics`) using a dedicated Lyrics API.
- **Audio Filters & Speed**: Apply real-time FFmpeg filters (`!filter bassboost, nightcore, karaoke, voice, etc.`) or adjust playback speed dynamically (`!speed 0.5 - 2.0`).
- **Smart Autoplay**: Toggle autoplay (`!autoplay [tag]`) to automatically fetch and play related music indefinitely without looping.
- **Clean Minimalist UI**: The `!player` (or `!np`) command displays an elegant interface featuring YouTube Music tags, subtitle availability, track upload dates, and dynamic `🔴 [LIVE]` tags.
- **Interactive Paginator**: View long music queues (`!queue`) and command lists (`!help`) cleanly through interactive multi-page embeds.
- **Advanced Playback Controls**: Full control over playback (`!pause`, `!resume`, `!stop`, `!seek`, `!forward`, `!back`, `!volume`).
- **Looping System**: Toggle looping for a single track (`!loop`) or the entire queue (`!loopqueue`). Perfectly handles skips and stops without losing queue continuity.
- **Queue Management**: Skip multiple tracks (`!skip [index]`), shuffle (`!shuffle`), rearrange (`!move`), remove (`!drop`), or clear (`!clear`) the queue.
- **Time Machine**: Instantly rewind to the previously played track without losing your current timeline (`!replay`).
- **DJ System**: Only the user who requested the first track (or a Server Admin) has permission to control the bot, but can pass the torch (`!transfer`).

## Requirements
- Python 3.9+
- **FFmpeg** must be installed on your operating system.
- Python packages (see `requirements.txt`):
  - `discord.py`
  - `yt-dlp`
  - `python-dotenv`
  - `PyNaCl`

## How to Run
1. Clone or download this repository.
2. Create a file named `.env` in the root directory and fill it with your Discord Bot Token:
   ```env
   BOT_TOKEN=your_bot_token_here
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Start the bot:
   ```bash
   python main.py
   ```

Alternatively, you can run the bot using Docker via `docker-compose up -d`.

**Default Prefix**: `!`
Type `!help` in a Discord channel to view the full list of available commands.
